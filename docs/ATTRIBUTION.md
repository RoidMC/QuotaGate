# QuotaGate Brand Usage Guidelines

## Overview

This document outlines the brand usage requirements for QuotaGate. These requirements are designed to protect the QuotaGate brand identity while allowing developers to use the software freely.

> 📌 **Basis**: These requirements are based on trademark and brand protection principles. They apply to users who incorporate QuotaGate's user interface components or distribute products that use QuotaGate as a visible platform.

---

## Attribution Requirements

### 1. Visual Attribution

#### 1.1 Required Attribution

If your application, product, or service **incorporates or uses QuotaGate's web UI components** (including but not limited to dashboards, admin panels, configuration interfaces, or any visual interface provided by QuotaGate), you must include the following attribution:

```
Powered by QuotaGate
```

The attribution must be displayed in at least one of the following locations:
- Application startup screen or splash screen
- About section or credits screen
- Settings or configuration screen
- Footer of web interfaces

#### 1.2 Display Requirements

The attribution must be:
- Clearly visible to end users
- In a readable font size (minimum 8pt or equivalent)
- Displayed for a reasonable duration if shown on a splash screen
- Not obscured by other elements

#### 1.3 Exemption for API-Only Usage

If your usage is **limited to programmatic API calls only** (i.e., you do not use any QuotaGate UI components, dashboards, or visual interfaces), you are **exempt** from the visual attribution requirement. You only need to comply with the MPL-2.0 license terms regarding copyright notices in source code.

### 2. Logo Usage

#### 2.1 Basic Requirements

If you choose to display the QuotaGate Logo (the "Logo"), you must strictly comply with the following terms:

- Use only the standard Logo (PNG/SVG) from the official QuotaGate website or authorized channels
- No modifications, distortions, or alterations are allowed
- Contact hi@roidmc.com for any questions or clarification

#### 2.2 Display Rules

- The logo must be displayed at an appropriate size and resolution
- The logo must not be modified, distorted, or altered in any way
- The logo must not be used in a way that suggests endorsement of your product by QuotaGate creators

#### 2.3 Prohibited Uses

Unless you have explicit prior written authorization from RoidMC Studios (via approval at hi@roidmc.com), you may not use the Logo in these scenarios:

1. **Implied Affiliation with RoidMC**
   - Claim or suggest your product is "officially developed by," "partnered with," or "endorsed by" RoidMC Studios
   - Use the Logo to imply a direct relationship with RoidMC (e.g., product names like "RoidMC Helper")

2. **Commercial Exploitation**
   - Use the Logo in ads, promotions, or sales materials to drive user acquisition or revenue
   - Feature the Logo in third-party content (blogs, tutorials, videos) to market your product

3. **Unauthorized Modification or Misrepresentation**
   - Modify the Logo (colors, proportions, design) to mimic RoidMC's official versions
   - Combine the Logo with other brands/logos as a joint identifier

4. **Misuse in Derivatives or Third-Party Platforms**
   - Use the Logo as a primary identifier for plugins, modules, or extensions based on QuotaGate
   - Display the Logo on non-technical platforms (e.g., corporate websites, investor pages) for brand association

#### 2.4 Ownership & Liability

- Logo IP (copyright, trademarks) belongs to RoidMC Studios. This grants you a limited, non-transferable display right only.
- Logo use does NOT mean RoidMC Studios endorses your product. RoidMC Studios is not liable for your product's issues.

---

### 3. Technical Limitations and Exemptions

If objective technical obstacles prevent the display of attribution as specified, you may apply for an exemption by contacting hi@roidmc.com.

Note: This exemption only waives the requirement to add visual text in your product's UI. MPL-2.0 obligations for the QuotaGate code (e.g., retaining copyright notices) still apply.

## Relationship to MPL-2.0

These brand usage guidelines are separate from the MPL-2.0 license. The MPL-2.0 license governs the use of QuotaGate source code, while these guidelines govern the use of the QuotaGate brand and visual identity.

Non-compliance with these guidelines may result in the termination of your brand usage authorization, but does not automatically terminate your MPL-2.0 license rights.
